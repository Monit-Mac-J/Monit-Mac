## 🔍 Overview

<a href="#monit">Monit</a> serves Mac users who want persistent system resource awareness — developers monitoring memory usage during application development, power users who want to know when their Mac is under load, and anyone who has opened Activity Monitor repeatedly to check what's consuming resources and wants that information available without that friction.

The <a href="#monit">developer use case</a> uses Monit to monitor memory and CPU behavior during development — watching memory usage climb during a memory leak investigation, monitoring CPU consumption during performance optimization, and maintaining awareness of system resource state during testing and debugging without switching to Activity Monitor repeatedly. The <a href="#monit">power user awareness use case</a> maintains constant awareness of system performance — knowing immediately when something is consuming unexpected CPU or memory, identifying background processes causing slowdowns, and having the system information needed to make informed decisions about closing applications or restarting services.

The <a href="#monit">network monitoring use case</a> uses Monit's throughput display to monitor download and upload activity — verifying that a large download is progressing, checking whether unexpected network usage is occurring during sensitive tasks, and monitoring network performance during remote work. <a href="#monit">Battery-conscious MacBook user</a> uses Monit's battery information for detailed battery health tracking — monitoring cycle count and capacity for awareness of battery aging over time.</p>

<a href="#monit">Configurable menu bar display</a> selects which metrics appear in the menu bar — CPU only, memory only, or combinations — without cluttering the menu bar with unwanted information. <a href="#monit">Refresh rate configuration</a> sets how frequently Monit updates its displayed values — more frequent updates for active monitoring, less frequent for background awareness with lower overhead.

<a href="#monit">Dark mode support</a> follows the macOS system appearance — menu bar display and popover interface adapt to light and dark mode automatically. <a href="#monit">Simple and focused design</a> — Monit covers the monitoring use cases most Mac users have without the complexity of full system monitoring suites, providing the right amount of information in the most accessible form.</p>

## ❓ FAQ

<details>
<summary>How is Monit different from Activity Monitor?</summary>

Activity Monitor is a full application requiring a window to be open. Monit provides the most commonly needed system metrics in the menu bar and a popover — always accessible without switching applications.

</details>

<details>
<summary>Can I set the refresh rate in Monit?</summary>

Yes. Refresh rate configuration adjusts how frequently Monit updates its displayed values.

</details>

<details>

Both — memory pressure (the macOS-level indicator of memory availability) and memory usage breakdown are displayed.
